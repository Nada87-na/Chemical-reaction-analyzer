
# 🧪 Chemical Reaction Analyzer (محلل التفاعلات الكيميائية)

## 💡 الفكرة (About the Project)
برنامج مكتوب بلغة بايثون مخصص لتحليل وتدقيق التفاعلات الكيميائية، موازنة المعادلات، وحساب الكتل المولية للعناصر والمركبات بشكل دقيق وسريع.

---

## 🛠️ الأدوات المستخدمة (Tech Stack)
* اللغة الأساسية: Python 3.10
* أداة الاختبار الذكي: Pytest (لملف test_analyzer.py)
* نظام الأتمتة والـ CI/CD: GitHub Actions (عبر ملف tests.yml)

---

## 🚀 طريقة التشغيل (How to Run)

1. تثبيت المتطلبات (Requirements):
  
   pip install -r requirements.txt
   
2. تشغيل البرنامج الأساسي:
  
   python main.py
   
3. تشغيل اختبارات التحليل الكيميائي:
  
   pytest test/test_analyzer.py
